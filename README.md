# clai

clai (pronounced "clay") is a command line ai assistant that uses LLM to answer to your queries.

## Installation

```bash
npm i @pr4j3sh/clai
```

## Usage

```bash
clai
```

- Run using

```bash
npm run dev
```

> By default `javascript` package is of type `cjs`. However, the final builds supports both, `cjs` and `esm`.
